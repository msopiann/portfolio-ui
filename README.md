## How to Clone

1. Clone repo:

```bash
git clone https://github.com/msopiann/portfolio-ui.git
cd portfolio-ui
```

2. Install dependensi:

```bash
bun install
```

3. Jalankan mode development:

```bash
bun run dev
```

Buka `http://localhost:3000`.

4. Build untuk produksi:

```bash
bun run build
```

5. Jalankan server produksi:

```bash
bun run start
```
